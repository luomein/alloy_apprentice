# `multiplicities` vs. `quantifiers`

## Comparison

|             | multiplicities  |  quantifiers                          |
| ----------- | -----------------|------------------- |
| Key Words | * set<br> * one<br> * some<br>* lone |  * all<br> * one<br> * some<br> * lone<br> * no |
| Expression       | `m sig S {}` <br> `sig S {f: m A}` <br> `sig S {f: A m -> n B}` | `( m x: S | F )` |
